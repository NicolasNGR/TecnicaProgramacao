## 🧠 Projetos em Destaque (Ciclo 3)

### 📍 Consulta de Endereço por CEP

Aplicação em Java que consome a **API ViaCEP** para buscar informações de endereço e manter um histórico de consultas.

**Principais conceitos:**

- Consumo de API REST
- Estruturas `switch` e `while`
- Uso de `ArrayList`
- Programação Orientada a Objetos

📂 Localização

```
Ciclo-3/ConsultaEndereco
```

---

### 📋 Sistema de Gestão de Tarefas (JDBC)

Sistema desenvolvido em **Java + JDBC + SQLite** para gerenciamento de tarefas com persistência em banco de dados.

**Principais conceitos:**

- JDBC
- SQLite
- Modelagem de dados
- Separação em camadas (Model, DAO, View)

📂 Localização

```
Ciclo-3/AtividadeJDBC
```
