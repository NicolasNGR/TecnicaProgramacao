# Cadastro de Produtos - Spring Boot

Projeto desenvolvido para prática de conceitos de **Spring Boot**, **Thymeleaf** e **MVC**, com foco na criação de um sistema simples de cadastro de produtos.

---

## Funcionalidades Implementadas

### Nível Básico
- Cadastro de produtos com:
  - Nome
  - Preço
  - Categoria
  - Estoque
- Estilização com CSS básico
- Exibição do total de produtos cadastrados

### Nível Intermediário
- Campo de seleção de categoria (Dropdown)
- Validação de preço negativo
- Remoção de produtos cadastrados

---

## Tecnologias Utilizadas

- Java 17
- Spring Boot
- Thymeleaf
- HTML5
- CSS3
- Maven
